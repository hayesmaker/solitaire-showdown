solitaire-showdown
==================

Project to rewrite Solitaire Showdown, The 1v1 multiplayer solitaire game which used to be on MSN Messenger.


Install:
--------

npm install

Run:
----

node app.js

To View:
--------

goto http://localhost:3000

Unit Testing:
-------------

karma start

or

goto http://localhost:3000/js/test/jasmine/SpecRunner.html
